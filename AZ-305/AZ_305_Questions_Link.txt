Enable simultaneous write operations in multiple Azure regions. --> Azure Cosmos DB
Ability to store JSON based items. --> Azure Cosmos DB
Support scaling up and down --> Service->Azure SQL Database, Service Tier->Hyperscale
Failover between replicas of the database must occur without any data loss. 
	--> Azure SQL Database Premium or Azure SQL Database Serverless 
You need to enable customer managed Transparent Data Encryption (TDE) for the instance. --> RSA 3072
The databases must be accessible if a datacenter fails. --> General Purpose 
Support a recovery time objective (RTO) of 15 minutes.Support a recovery point objective (RPO) of 24 hours.
	--Azure Site Recovery
Support geo-replication, Store and access data relationally. --> PostgreSQL
You have an app named App1 that uses two on-premises Microsoft SQL Server databases named DB1 and DB2.
	-Support server-side transactions across DB1 and DB2.
	--ANS:two databases on the same Azure SQL managed instance

	-an elastic pool that contains 20 Azure SQL databases
